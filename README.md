# ACT Website - Final Project

## Description
This is a website for American College of Technology (ACT) created as the final project for Introduction to Web Development. It has 6 pages with information about ACT's programs and facilities.

## Features
- 6 responsive web pages (Home, About, Programs, Academy, Why ACT, Contact)
- Mobile-friendly design with Flexbox and CSS Grid
- Contact form with JavaScript validation
- Clean and organized code

## Technologies Used
- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript
- Git & GitHub

## How to Run
1. Download the project files
2. Open `index.html` in any web browser
3. Use the navigation menu to explore all pages
